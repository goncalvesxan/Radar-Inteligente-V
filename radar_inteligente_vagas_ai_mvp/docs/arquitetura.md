# Arquitetura recomendada

## Princípio
Copiloto, não bot. A ferramenta prepara a candidatura, mas o usuário revisa e envia.

## Pipeline
1. Entrada de vagas por CSV, alerta manual, e-mail ou link colado.
2. IA/algoritmo calcula score de aderência.
3. Sistema gera mensagem, nota de candidatura e palavras-chave.
4. Usuário abre site externo.
5. Autofill supervisionado preenche campos objetivos.
6. Usuário revisa e envia.
7. Pipeline registra histórico.

## Evoluções futuras
- Integração com Gmail para capturar alertas de vagas.
- Integração com Google Sheets.
- Dashboard Power BI.
- Adaptação de currículo por vaga.
- Biblioteca de respostas para perguntas eliminatórias.
